# font_transfer
字体反爬通用转换工具，采用OCR识别技术，秒杀静态或动态的字体文件。

本算法可以自适应让字型居中，提升ocr识别的准确率；

本算法可以自适应图片的大小，不用操心字体文件中的字型过多的问题。

## Python版本<=3.8

## 安装环境

cnocr版本

```shell
pip install -r requirements_cnocr.txt
```

muggle-ocr版本

```shell
pip install -r requirements_muggle_ocr.txt
```
